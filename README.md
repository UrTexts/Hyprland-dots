## рЯЪА Install

```bash
git clone https://github.com/UrTexts/Hyprland-dots.git ~/dotfiles
cd ~/dotfiles
bash install.sh
```

> [!NOTE]
> Works on **Arch Linux** and **Fedora**. The script auto-detects your distro.

> [!TIP]
> Existing configs are backed up as `.bak` files вАФ nothing gets deleted.

The install script handles:

- вЬЕ Package installation (pacman or dnf)
- вЬЕ Symlinking all configs to `~/.config/`
- вЬЕ Everforest cursor theme installation
- вЬЕ PipeWire service activation
- вЬЕ Starship added to `.zshrc`

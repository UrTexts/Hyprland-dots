## ЁЯЪА Install

```bash
git clone https://github.com/UrTexts/Hyprland-dots.git ~/dotfiles
cd ~/dotfiles
bash install.sh
```

> [!NOTE]
> Works on **Arch Linux** and **Fedora**. The script auto-detects your distro.

> [!TIP]
> Existing configs are backed up as `.bak` files тАФ nothing gets deleted.

The install script handles:

- тЬЕ Package installation (pacman or dnf)
- тЬЕ Symlinking all configs to `~/.config/`
- тЬЕ Everforest cursor theme installation
- тЬЕ PipeWire service activation
- тЬЕ Starship added to `.zshrc`

## ЁЯФТ Lock Screen

Hyprlock is configured with a blurred Ghibli wallpaper, JetBrains Mono clock, and a minimal password field.

To lock manually:

```bash
hyprlock
```

If you're using the `matugen-minimal` SDDM theme, set the wallpaper with:
